This directory should contain the extension icons.

Required icon sizes:
- icon-16.png (16x16) - Favicon, tab icon
- icon-32.png (32x32) - Windows taskbar
- icon-48.png (48x48) - Extension management page
- icon-128.png (128x128) - Chrome Web Store, installation

You can generate these from a single source image using tools like:
- ImageMagick: convert icon.png -resize 16x16 icon-16.png
- Online tools: https://favicon.io/

Recommended: Create an SVG source file and export to PNG at each size.

For development, you can use placeholder images. The extension will work without icons.
